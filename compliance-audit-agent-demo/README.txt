全自动合规审计与政策影响分析 Agent

这是一个无需安装依赖、可直接打开运行的中文单页网页 Demo。

打开方式
1. 进入当前文件夹
2. 双击 index.html
3. 点击“开始合规审计”查看完整流程

功能说明
- 使用内置政策样本和业务文档样本演示合规审计
- 展示 4 个 Agent 的协作流程
- 自动生成强制性条款图谱
- 自动识别合同/产品/制度中的潜在冲突
- 输出长链推理结果、风险矩阵、整改建议与整改后残余风险
- 支持上传 .txt / .json 文件或直接粘贴文本
- 支持导出文本报告

项目结构
- index.html：页面结构
- style.css：界面样式
- script.js：审计逻辑与交互
- sample-data/policies.json：政策样本
- sample-data/business-docs.json：业务文档样本
